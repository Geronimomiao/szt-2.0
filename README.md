**项目简介**

* 基于 ColorUI 构建的小程序 分教师端 学生端

**项目目录结构**

```
|
|- component // 可复用组件
|
|- lib // 第三方库
|    |- ColorUI // UI框架
|
|- utils
|    |- api.js // 后台 api 
|    |- router.js // 前端 路由
|    |- store.js // 存一些关键的常量  
|
|- pages
|    |- launch // 启动页图 
|    |- select // 选择身份
|    |- video // 情景再现
|
|
|- static
|    |- css 
|        |- common.css // 覆盖第三方 默认样式
|
|- app.js // 全局变量 方法
|
|- app.wxss // 公共样式
```
